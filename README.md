# 46-竞赛管理系统

[文档地址](http://wechat.zjrcsy.cn/)

##### 技术栈:springBoot+Vue+Mysql+MyBatis



##### 用户端: 

1.首页:轮播图展示、公告信息展示

2.竞赛项目：竞赛项目列表展示，用户若自己为团队队长，可进行报名操作

3.团队信息：展示团队列表，可查看团队详情信息

4.团队组建：邀请他人组队，展示（我发起的申请、收到的申请、我的团队）

5.个人中心：修改个人信息、查看竞赛报名列表、竞赛指导、申请退款的项目、我的收藏



**教师端：**

1.个人中心：个人信息修改及密码修改

2.竞赛项目：查看竞赛项目详情，查看参赛报名团队以及竞赛结果

3.团队信息：查看团队信息，查看反馈

4.竞赛指导：可对已报名的竞赛团队进行意见指导和删除



##### 管理员端: 

1.个人中心：个人信息修改及密码修改

2.学生管理：对用户的增删改查

2.教师管理: 对教师的增删改查

3.学生画像选项:  对学生画像选项进行操作

4.竞赛项目：查看竞赛项目详情，查看参赛报名团队以及竞赛结果，可进行增删改查操作

5.竞赛类型：增删改查操作

6.团队信息：查看团队信息，查看反馈，可进行改删操作

7.竞赛指导：查看教师对竞赛团队的指导意见

8.申请退款：对退款申请进行操作

9.系统管理：轮播图管理、公告信息管理、公告信息分类管理、系统日志查看
